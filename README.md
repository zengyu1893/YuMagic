# YuMagic（玉玉画布）

AI 视觉创作、节点工作流与素材管理工具。

## API 接口与安全

推荐的两个 API 基础地址：

- [https://yuli.host](https://yuli.host)
- [https://ai.yuliapi.com](https://ai.yuliapi.com)

> API Key 只保存在本机设置或环境变量中，不要提交 `.env`、设置 JSON、日志、上传文件或任何真实凭据。

---

## 项目简介

YuMagic 是一款本地优先的 AI 视觉创作工作台，集图像生成、图像编辑、视频生成、节点工作流、创意库和素材管理于一体。

生成的图片、视频和任务结果可以直接保存到本地，并在桌面工作区或节点画布中继续整理、编辑和复用。

本项目基于企鹅工坊（Penguin Magic）继续开发。

## 主要功能

### 桌面素材管理

- 使用桌面式工作区管理图片和其他创作素材
- 支持拖放、移动、多选和批量操作
- 支持文件夹、素材分组和叠放整理
- 生成结果可以直接保存到本地工作区

### 节点画布

- 提供可自由缩放和移动的节点画布
- 使用节点连接图像、文本、视频和工作流
- 支持执行完整的 AI 创作流程
- 支持查看节点状态、任务进度和生成结果
- 支持对生成结果继续编辑和再次生成

### AI 图像生成与编辑

- 支持文本生成图片
- 支持参考图、图生图和图片编辑
- 支持尺寸、比例、分辨率和模型选择
- 支持多张图片和批量任务
- 生成结果自动保存到本地

### AI 视频生成

- 支持文本生成视频和图片生成视频
- 支持不同视频服务商和模型
- 支持视频任务提交、状态查询和结果保存
- 支持按服务商配置独立的 API 地址和参数

### 文本 AI 与多角度生成

- 支持文本提示词处理和内容生成
- 支持通过参考图片生成不同视角
- 支持将文本结果继续传递给图像或视频节点

### 创意库

- 管理和复用常用提示词与创意模板
- 支持搜索、预览和快速应用创意
- 支持智能导入创意内容
- 可从 [Awesome Prompt Gallery](https://opennana.com/awesome-prompt-gallery/) 获取编号
- 导入路径：`创意库` → `智能导入` → `输入编号`

### API 服务商设置

- 支持配置多个 API 服务商
- 支持独立设置 API 基础地址和 API Key
- 支持服务商配置切换和模型选择
- 支持 RunningHub 工作流配置
- API Key 仅保存在本机，不应上传到 GitHub

### 任务与历史记录

- 支持查看任务队列和执行进度
- 支持管理图片、视频和工作流任务
- 支持查看历史生成结果
- 支持失败任务重试和结果恢复

### 本地数据存储

- 正式数据统一保存在项目的 `data/` 目录
- 通过本地 Node.js 与 Express 服务读写数据
- 图片、视频、配置和任务记录优先保存在本机
- 使用前请定期备份重要数据

## 快速开始

### 环境要求

- Windows 10 或 Windows 11
- Node.js 18 或更高版本
- npm

### 一键安装与启动

首次使用时，双击运行：

```text
install.bat
```

安装完成后，双击运行：

```text
start.bat
```

启动成功后，在浏览器打开：

[http://127.0.0.1:8765](http://127.0.0.1:8765)

项目更新后，建议重新运行 `install.bat`，确保依赖已经更新。

### 手动安装与启动

安装前端依赖：

```bash
npm install
```

安装后端依赖：

```bash
cd backend-nodejs
npm install
cd ..
```

构建前端：

```bash
npm run build
```

启动后端服务：

```bash
cd backend-nodejs
node src/server.js
```

然后在浏览器打开：

[http://127.0.0.1:8765](http://127.0.0.1:8765)

## API 配置

1. 启动 YuMagic。
2. 打开设置。
3. 进入 API 或服务商设置。
4. 选择需要使用的服务商。
5. 填写 API 基础地址和 API Key。
6. 保存配置并选择对应模型。

推荐的 API 基础地址：

- [https://yuli.host](https://yuli.host)
- [https://ai.yuliapi.com](https://ai.yuliapi.com)

请勿将以下内容上传到公开仓库：

- `.env` 文件
- 包含 API Key 的设置 JSON
- 调试日志和错误日志
- 用户上传的图片、视频或其他文件
- 任务记录和历史记录
- Cookie、Token 或任何真实凭据

提交代码前，建议检查：

```bash
git status
git diff --cached
```

## 数据与隐私

YuMagic 采用本地优先的存储方式。项目本身不会要求你把 API Key 提交到 GitHub。

使用公开仓库时，请确认 `.gitignore` 已排除本地配置、日志、上传文件、生成结果和其他敏感数据。

如果 API Key 曾经被提交到 Git 历史中，仅删除文件并不能保证安全。请立即前往对应服务商后台撤销旧 Key，并生成新的 Key。
